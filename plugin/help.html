Eepsite Key & Hostname Registration

Your Base 32 address is: $B32. Others may access your eepsite using this address, even if you do not publish a hostname. Once you decide on a hostname, you may add the key to your local addressbook.

Your Base 64 key is:
$B64

To generate a registration authentication string for your hostname (required for hostname registration at stats.i2p), run the following from the command line:
java -jar $I2P/lib/i2p.jar privatekeyfile -a your-hostname.i2p $PLUGIN/eepPriv.dat

Your private key file is located at: $PLUGIN/eepPriv.dat
Make sure you backup this file and move the copy outside of $PLUGIN/ as it controls ownership of your B64/B32 address.

Configuration & Customization

  • Your eepsite tunnel is configured for 2 inbound and 2 outbound tunnels, 3 hops each. To change the tunnel settings, edit $PLUGIN/i2ptunnel.config and then restart the plugin from the Plugin Manager (the tunnel will not appear in the Tunnel Manager). If your tracker gets over 1000 peers, you will probably want to increase the number of tunnels.
  • The only configuration for ZzzOT itself is the interval, which tells the clients how long to wait between announces. The default is 1620 seconds (27 minutes). Note: The easiest way to reduce the load on your tracker is to increase the interval. To do so, edit the file $PLUGIN/zzzot.config, then stop and restart the plugin.
  • The Jetty webserver port is 7662. If you must change it, edit jetty.xml, i2ptunnel.config, and plugins.config in the directory $PLUGIN, then stop and restart the plugin.
  • Your eepsite home page is located at: $PLUGIN/eepsite/docroot/index.html, you may edit it to add information about your tracker or links to additional content.
  • Your eepsite document root is: $PLUGIN/eepsite/docroot/. You may put other files there if you wish to host additional content on your eepsite.
  • To change the display name for the site (for the logo and page titles), add the line: sitename=mytracker to $PLUGIN/zzzot.config (substituting mytracker with your desired name) and then restart the plugin.
  • If you wish to change the appearance of the site, edit the css file located at: $PLUGIN/eepsite/docroot/zzzot.css.

As you probably know, an open tracker does not require torrents to be registered, and it does not host torrent files. However, you can host torrent files elsewhere on the eepsite, for example at /torrents.

Note: Directory listings are enabled on the server, so you don't need create a web page to serve files; content in any folders you create in $PLUGIN/eepsite/docroot/ will be immediately available to your users.
